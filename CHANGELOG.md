# Changelog

## 2026.04.26-release.1

- Added public static release packaging under `dist/`.
- Added cache-busted stylesheet and module entry URLs plus a shared `BUILD_VERSION` constant.
- Added mobile responsive layout, safe-area handling, and touch controls.
- Added loading screen with progress/status text.
- Added fullscreen toggle button.
- Added reset-save confirmation for localStorage data.
- Added privacy-friendly analytics placeholder disabled by default.
- Added disabled platform SDK adapter placeholders for CrazyGames, Poki, itch.io, and GameDistribution.
- Added publishing checklist and release documentation.
- Added MIT license.

## Content Expansion Baseline

- Expanded Story Mode to seven stages with wave-based progression.
- Added hazards, enemy variants, upgrades, Scrap currency, unlocks, stats, and menu updates.
- Added local co-op support, weapons, supers, encyclopedia screens, options persistence, and debug overlay.
